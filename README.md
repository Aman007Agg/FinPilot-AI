# FinPilot AI
AI Powered Personal Finance Management Platform


## 📖 Overview

FinPilot AI is a modern enterprise-level personal finance management platform built using Angular, ASP.NET Core, FastAPI, SQL Server, Docker, Kubernetes, and Azure.

The platform helps users:

- Track expenses
- Manage budgets
- View dashboards
- Get AI-powered financial insights
- Categorize expenses automatically
- Chat with an AI assistant

## 🏗 Technology Stack

### Frontend
- Angular 20
- TypeScript
- Angular Material

### Backend
- ASP.NET Core 9 Web API
- Entity Framework Core

### AI Service
- Python
- FastAPI
- LangChain
- OpenAI / Groq

### Database
- SQL Server

### DevOps
- Docker
- Kubernetes
- Azure
